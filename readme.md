# 📄 Resume Screener

An automated tool designed for smart resume analysis. It leverages Artificial Intelligence to extract key skills, experience summaries, and hiring recommendations from PDF resumes. The project runs entirely locally using **Ollama**, ensuring data privacy and zero API costs.

## 📸 Architecture Overview

## 🛠 Tech Stack

- **Backend:** Python, FastAPI, PyMuPDF (fitz)
- **Frontend:** Next.js, React, Tailwind CSS, TanStack Query
- **AI Engine:** Ollama (default model: `llama3.2`)
